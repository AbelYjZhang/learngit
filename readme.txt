```git
git发生错误之回退版本序列
git reset --hard 29160132824f5c2eb0747ee789cc75a43c302f90             ##（HEAD is now at 2916013 加入百度统计ID）
git push -f -u origin master &&  git push -f -u master &&  git push -f -u     ##（定位到当前消失的目录文件夹）
git push                           ##（出现原先设置效果）
```
